# Molduras Lions 2.0

Versão visual renovada do site público e do painel administrativo.

## Principais mudanças no Index

- Hero mais limpo, sem mensagens sobre privacidade ou servidores.
- Destaque `✨ Simples e rápido`.
- Informação `📱💻 Funciona no celular e computador`.
- Favoritas com coração vermelho.
- Correção do duplo botão “x” no campo de pesquisa.
- Comparação Antes/Depois movida para Ajustes avançados.
- Refinamento visual de cartões, barra de pesquisa, editor e rodapé.

## Principais mudanças no Admin

- Favicon com o logo do Lions.
- Hero e dashboard redesenhados.
- Cartões, navegação, ferramentas e estados de publicação refinados.
- Correção visual do botão nativo de limpar pesquisa duplicado.
- Funcionamento e integração com GitHub preservados.
