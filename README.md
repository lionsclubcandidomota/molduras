# V6.7 — Controle global de categorias

- Botões **Expandir todas** e **Contrair todas** no Index.
- **Expandir todas** abre as categorias e exibe todas as molduras.
- **Contrair todas** fecha todas as categorias visíveis.
- Ao clicar em uma categoria específica, ela é aberta e expandida automaticamente.
- O estado continua salvo no navegador.
- Durante uma pesquisa, os controles globais ficam indisponíveis porque os resultados já são abertos automaticamente.
