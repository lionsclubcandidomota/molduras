# Molduras Lions v13 — produtividade

## Site público
- salvamento automático local da moldura e dos ajustes
- recuperação do último trabalho
- favoritos sem cadastro
- lista de molduras recentes
- busca sem acentos preservada

## Admin
- editor lateral ao editar uma moldura
- validação em tempo real para nome, ID e categoria
- bloqueio preventivo de IDs duplicados
- barra fixa de alterações pendentes
- reordenação por arrastar e soltar preservada

Substitua os arquivos das pastas `site-publico` e `admin`. Preserve seu `molduras.js` e a pasta de imagens do repositório.
